Aclaracion de controles de juego:
1 se guarda el juego
2 se carga el juego
P Pantalla de pausa
I Inventario
Movimiento (teclas por defecto) A-W-S-D

En el video se puede observar que con ests mismos botones se realizan las acciones previamente mencionadas.
Además de lo siguiente:Las particulas, sistema de barra de vida, sistema de daño, sistema de inventario, sistema de menus, cambio de graficos(bajo, medio y alto), enemigos tienen Funcionalidad de patrullaje(de forma temporal antes de más mejoras).
